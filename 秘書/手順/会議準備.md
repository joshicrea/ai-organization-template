# 会議準備・議事録

トリガー: 「会議準備」「議事録」

## 会議前の準備

1. Google Calendarから該当イベントの詳細を取得
2. 参加者、議題、関連資料を整理
3. 以下のフォーマットで準備メモを作成:

```
## {会議名} - {日付}
- 時間: {開始-終了}
- 参加者: {名前リスト}
- 場所/URL: {場所}

### 議題
1. {議題}

### 事前確認事項
- {確認すべきこと}

### 持参資料
- {必要な資料}
```

## 議事録

会議後に「議事録」と言われたら以下を作成:

```
## {会議名} 議事録 - {日付}

### 参加者
{名前リスト}

### 決定事項
1. {決まったこと} → 担当: {誰} / 期限: {いつまで}

### 次のアクション
- [ ] {タスク} - {担当者} - {期限}

### メモ
{補足情報}
```

保存先: 秘書/記録/会議/YYYY-MM.md に追記（月単位）
