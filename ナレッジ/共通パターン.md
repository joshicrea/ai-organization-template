# 共通パターンナレッジ

## コピーの3原則（全成果物に適用）

| 原則 | 内容 |
|------|------|
| 減らす | 削っても意味が変わらない語は削る。目標は初案の半分 |
| 具体化する | 「しっかり」「きめ細かく」→ 数値・プロセス・事例に |
| 単純にする | 1文1メッセージ。主語と述語を近づける。結論を先に |

## 削ってよい語・表現

| カテゴリ | 例 |
|----------|-----|
| 接続・前置き | 「〜ということになります」「〜ということが言えます」 |
| 冗長 | 「〜することが可能です」→「〜できます」 |
| 重複 | 「まず最初に」「いずれか一つ」 |
| 弱い修飾 | 「少し」「ある程度」「かなり」→ 具体数値に |

## 曖昧ワード変換表

| 曖昧 | 具体化の方向 |
|------|-------------|
| しっかり | 頻度・基準・成果物 |
| きめ細かく | プロセス・ステップ・チェック項目 |
| 安心 | 保証内容・実績・第三者評価・数字 |
| 柔軟に | 納期・変更可能な範囲・対応時間 |
| 充実 | 何がどう増えるか（機能・回数・時間） |
| 〜など | 主要なものは省略せず書く |

## お客様の声フォーマット（LP・セミナー共通）

```
見出し: 数字を含む成果（月商xx万、xx倍、xx月で達成等）
名前: {名前} さま
肩書: 役割/立場/属性
声: 自然な文章。トーン・エピソード・感情を保持する
```

数字がない場合は、本文から最もインパクトのある変化・感情を抽出して見出しにする。

## デザイン共通ルール

- 配色: 3色以内（ベース黒・メイン・アクセント）
- フォント: メイリオで統一、1種類のみ
- サイズ: 本文18pt以上、タイトル32pt
- 装飾: 太字と下線のみ（斜体・影は使わない）
- 写真: 外国人の写真・アイコンは使わない。イメージ写真は入れない
- 背景: 白のまま。形は四角と丸に絞る
