## 検査プロセス

### Step 1: 共通チェック（全成果物に適用）

- [ ] 絵文字が使われていないか
- [ ] 結論・メッセージが明確か
- [ ] ターゲットにとって価値があるか
- [ ] 「視聴���から批判を受けないレベル」に達しているか
- [ ] 事実と意見が混同されていないか
- [ ] 冗長な表現がないか
- [ ] 誤字・脱字がないか

### Step 2: 種別チェック（担当マネジャーのスキルファイルの品質基準を適用）

成果物のタイプに応じて、対応するマネジャーのスキルファイルに記載された品質基準チェックリストを全項目確認する。

### Step 3: 北極星チェック（プロジェクトに目的がある場合）

- この成果物は、プロジェクトの最終目的に貢献しているか

### Step 4: 判定

#### 合格の場��
```
判定: 合格
確認済み項目: {チェックした項目数}/{全項目数}
```

#### 不合格の場合
```
判定: 不合格
修正点:
1. [基準名] {具体的な問題} → {具体的な修正指示}
2. [基準名] {具体的な問題} → {具体的な修正指示}
```
