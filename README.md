# AI組織テンプレート

Claude Code をAI組織の「司令塔」として使うためのテンプレート。
タスクを指示するだけで、適切なAIエージェント（部署）が自動で仕事をこなす。

## できること

| 部署 | できること |
|---|---|
| 記事執筆 | ブログ・note・Kindle原稿 |
| メルマガ | メルマガ・ステップメール・LINE配信文 |
| スライド制作 | プレゼン資料・セミナースライド |
| LP構築 | ランディングページの構築・改修 |
| 市場リサーチ | 市場調査・競合分析 |
| スクリプト開発 | 自動化ツール・スクリプト開発 |
| 動画制作 | 動画企画・台本・VSL・編集指示 |
| BtoB提案 | 提案書の構成・コピー |
| 秘書 | メール・カレンダー・チャット・タスク管理 |
| 品質管理 | 成果物の品質チェック（自動） |

## セットアップ

### 必要なもの

- Windows 10/11
- Anthropic APIキー（Claude Code用）

### 手順

**1. ダウンロード**

```powershell
git clone https://github.com/joshicrea/ai-organization-template.git ai-organization
cd ai-organization
```

> git が入っていない場合は「Code」→「Download ZIP」でもOK。

**2. セットアップを実行**

```powershell
.\setup.bat
```

名前・肩書き・事業内容などを入力。Node.js と Claude Code は自動インストール。

**3. Claude Code を起動**

```powershell
claude
```

あとは話しかけるだけ。

## 使い方

```
「セミナー用のスライド原稿を作って」
「この商品のLP構成を考えて」
「競合3社を調べてまとめて」
「おはよう」        → 朝ブリーフィング
「メール見て」      → Gmail要対応抽出
「状況」            → 全案件の進捗レビュー
「ブログ記事を書いて」
```

## フォルダ構成

```
ai-organization/
├── CLAUDE.md              # 司令塔プロトコル
├── setup.bat / setup.ps1  # セットアップ
├── 組織/
│   ├── 組織図.md          # 部署一覧とルーティングルール
│   └── スキル/            # 部署別スキルファイル（10部署）
├── ナレッジ/              # エラーから学んだ知見（自動蓄積）
├── プロジェクト/          # 事業情報・接続情報
└── 秘書/                  # 秘書の詳細プロトコル
    ├── 手順/              # 朝ブリーフィング、メールトリアージ等
    ├── 案件/              # 進行中の案件（自動管理）
    ├── 記録/              # ブリーフィング記録等
    ├── 資料/              # リサーチ結果
    └── 完了/              # 完了した案件
```

## カスタマイズ

### 部署を追加する

1. `組織/スキル/{新しい部署名}.md` を作成
2. `組織/組織図.md` のルーティングテーブルに追加

### 既存の部署を調整する

`組織/スキル/` 内の該当ファイルを編集するだけ。

### ナレッジの仕組み

エラーや修正が発生すると、自動で `ナレッジ/` に記録される。
次回以降、同じ部署が起動するときにナレッジを読み込むので、同じミスを繰り返さない。
