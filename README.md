# AI Organization Template

Claude Code をAI組織の「司令塔」として使うためのテンプレート。
タスクを指示するだけで、適切なAIエージェント（部署）が自動で仕事をこなす。

## できること

| 部署 | できること |
|---|---|
| 記事執筆 | ブログ・note・Kindle原稿 |
| メルマガ | メルマガ・ステップメール・LINE配信文 |
| スライド制作 | プレゼン資料・セミナースライド |
| LP構築 | ランディングページの構築・改修 |
| 市場リサーチ | 市場調査・競合分析 |
| スクリプト開発 | 自動化ツール・スクリプト開発 |
| 動画制作 | 動画企画・台本・VSL・編集指示 |
| BtoB提案 | 提案書の構成・コピー |
| 秘書 | メール・カレンダー・チャット管理 |
| 品質管理 | 成果物の品質チェック（自動） |

## ダウンロードとセットアップ

### 必要なもの

- Windows 10/11
- Anthropic API キー（Claude Code用）

### 手順

**1. ダウンロード**

PowerShell またはターミナルを開いて以下を実行:

```powershell
git clone https://github.com/joshicrea/ai-organization-template.git ai-organization
cd ai-organization
```

> git が入っていない場合は、GitHub のページから「Code」→「Download ZIP」でもOK。
> 解凍したフォルダ名を `ai-organization` に変更してください。

**2. セットアップを実行**

```powershell
.\setup.bat
```

対話形式で以下を聞かれるので入力:
- 名前
- 肩書き
- 事業形態
- 事業内容
- 作業フォルダのパス
- Obsidian vault のパス（使っていなければEnterでスキップ）

Node.js と Claude Code が入っていなければ自動でインストールされる。

**3. Claude Code を起動**

```powershell
claude
```

これで準備完了。あとは話しかけるだけ。

## 使い方

Claude Code を起動した状態で、普通に指示するだけ:

```
「セミナー用のスライド原稿を作って」
「この商品のLP構成を考えて」
「競合3社を調べてまとめて」
「明日の予定を確認して」
「ブログ記事を書いて」
```

指示を受けると:
1. 司令塔（Claude Code）が適切な部署を自動選定
2. 部署のスキルファイルを読み込んでエージェントを起動
3. 成果物が返ってきたら品質チェック
4. 問題があれば自動修正、なければ納品

## フォルダ構成

```
ai-organization/
├── CLAUDE.md              # 司令塔プロトコル（これが頭脳）
├── setup.bat              # セットアップ起動用
├── setup.ps1              # セットアップ本体
├── 組織/
│   ├── 組織図.md          # 部署一覧とルーティングルール
│   └── スキル/            # 部署別スキルファイル（10部署）
├── ナレッジ/              # エラーから学んだ知見（自動蓄積）
└── プロジェクト/          # プロジェクト別の情報
    ├── 業務概要.md        # あなたの事業情報
    └── 接続情報.md        # 外部サービスの接続先
```

## カスタマイズ

### 部署を追加する

1. `組織/スキル/{新しい部署名}.md` を作成
2. `組織/組織図.md` のルーティングテーブルに追加

### 既存の部署を調整する

`組織/スキル/` 内の該当ファイルを編集するだけ。
スキルファイルがそのままエージェントへの指示書になる。

### ナレッジの仕組み

エラーや修正が発生すると、自動で `ナレッジ/` に記録される。
次回以降、同じ部署が起動するときにナレッジを読み込むので、同じミスを繰り返さない。
